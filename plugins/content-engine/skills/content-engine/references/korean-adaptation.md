# Korean Content Adaptation

## Core Principle

Korean content adaptation is NOT translation. It is cultural rewriting — adapting the idea, tone, structure, and references for a Korean-native audience. A translated LinkedIn post reads like a foreign document. An adapted one reads like it was written by a Korean founder.

## Korean Platform Landscape

### Key Platforms

| Platform | Korean Equivalent | Content Type | Audience |
|----------|------------------|--------------|----------|
| Google Search | Naver Search + Blog | SEO + blog content | Everyone |
| Medium/Substack | Naver Blog + Brunch | Long-form content | Professionals, creators |
| Twitter/X | Twitter KR + Threads | Short-form, opinions | Tech, startup, media |
| Facebook Groups | Naver Cafe + KakaoTalk Groups | Community discussion | Niche communities |
| Email Newsletter | KakaoTalk Channel | Direct messaging | B2C, local business |
| LinkedIn | LinkedIn KR (growing) | Professional content | Corporates, HR, execs |

### Platform Priority for Korean Market

1. **Naver Blog** — The starting point. Naver search is still dominant for discovery.
2. **KakaoTalk Channel** — Direct reach to users (Korea's WhatsApp equivalent, used by 95%+ of population)
3. **Naver Cafe** — Community-based engagement and trust building
4. **Brunch** — Premium long-form content (Kakao's publishing platform)
5. **Instagram** — Visual content, especially for B2C
6. **Twitter KR** — Growing among tech/startup audience
7. **LinkedIn KR** — Emerging for B2B, HR, executive audience

## Naver Blog

### Platform Character

- Naver Blog is the foundation of Korean online content discovery
- Naver search prioritizes Naver Blog results over websites
- LLM-based ranking (2024+) rewards comprehensive, well-structured content
- C-Rank (Creator Rank) determines visibility — authority builds over time
- DIA (Direct Intelligence Algorithm) evaluates content quality
- Keyword-rich, well-structured content with original images ranks best

### Format Specs

| Element | Spec | Notes |
|---------|------|-------|
| Title | 30-40 chars (Korean) | Include primary keyword in first half |
| Body length | 1,500-3,000 chars | Detailed but scannable |
| Images | 5-10 per post | Original images preferred over stock |
| Headings | Use Naver's heading styles (제목, 소제목) | Critical for readability |
| Tags | 5-10 tags | Include both broad and niche tags |
| Sections | Use 구분선 (divider lines) between sections | Improves readability score |

### Naver Blog Post Template

```
제목: [키워드] [구체적 결과/방법] — [숫자나 특정 세부사항 포함]
  예시: "1인 개발자 SaaS 런칭 3개월 만에 MRR 500만원 달성한 방법"

서론 (100-150자):
  — 공감 포인트로 시작 (독자의 고민/상황 묘사)
  — "혹시 ~하고 계신가요?"
  — "저도 처음에는 ~했습니다"
  — 이 글에서 얻을 수 있는 가치 약속

본문 (1,000-2,000자):
  소제목 1: [첫 번째 핵심 포인트]
    — 구체적 설명 + 실제 경험
    — 관련 이미지 (스크린샷, 과정 사진)

  소제목 2: [두 번째 핵심 포인트]
    — 데이터나 수치 포함
    — 비교 테이블이나 리스트 활용

  소제목 3: [세 번째 핵심 포인트]
    — 실행 가능한 조언
    — 도구나 리소스 추천

결론 (100-150자):
  — 핵심 요약 (3줄 이내)
  — "도움이 되셨다면 이웃추가 부탁드려요"
  — 관련 글 링크 (내부 링크)

태그: #키워드1 #키워드2 ... (5-10개)
```

### Naver Blog SEO

- **Title**: Primary keyword must appear in the first half
- **첫 문단**: Include keywords naturally in the opening paragraph
- **Image alt text**: Naver indexes image descriptions
- **길이**: Longer posts (1,500+ chars) rank better than short posts
- **업데이트**: Regularly updating old posts boosts ranking
- **이웃/댓글**: Engagement (neighbor adds, comments) improves C-Rank
- **원본 콘텐츠**: Original content is strongly preferred over reposts

## KakaoTalk Channel

### Platform Character

- Direct messaging to opted-in users
- 95%+ of Koreans use KakaoTalk daily
- Messages feel personal — respect the inbox
- Rich message formats: text, image, carousel, button
- Frequency: 1-3 messages per week maximum (more = mass unsubscribe)

### Message Specs

| Format | Spec | Notes |
|--------|------|-------|
| Text message | Max 1,000 chars | Short, focused, one CTA |
| Image message | Image + text + button | Most common format |
| Carousel | Up to 6 cards | Product showcases, tips |
| Wide image | 800x400px header + text | Event announcements |
| Button types | Web link, chat, phone | Max 2 buttons per message |

### KakaoTalk Message Template

```
[이미지: 800x400px — 핵심 메시지 포함]

📌 [핵심 제목 — 15자 이내]

[2-3줄 본문]:
  — 핵심 가치/혜택
  — 구체적 수치나 결과
  — 긴급성 요소 (진짜인 경우만)

[버튼 1: 주요 CTA — "자세히 보기" / "신청하기" / "무료 체험하기"]
[버튼 2: 보조 CTA — "문의하기" (선택)]
```

### KakaoTalk Content Rules

- **존댓말**: Always use 해요체 (B2C) or 합니다체 (B2B)
- **이모지**: Use sparingly — 1-2 per message max (not like Instagram)
- **시간**: Best sending time: 화-목 오전 10-11시, 오후 2-3시
- **빈도**: Max 2-3 messages per week. More = unsubscribe
- **개인화**: Use 이름 (name) variable when possible

## Brunch (브런치)

### Platform Character

- Kakao's premium content platform (like Korean Medium)
- Curated — requires application to become a writer
- Long-form, essay-style content
- Clean reading experience, no ads
- Strong among professionals, creatives, and entrepreneurs
- Great for building authority and thought leadership

### Brunch Post Template

```
제목: 에세이 스타일 — 호기심 유발하는 문장
  예시: "3년 동안 회사를 다니며 몰래 만든 서비스가 월 매출 1억을 넘기까지"

본문:
  — 개인적 이야기로 시작 (공감)
  — 경험 + 인사이트 + 교훈
  — 2,000-5,000자 권장
  — 이미지 삽입 (분위기 맞는 사진, 데이터 시각화)
  — 문단 사이 여백 충분히
  — 결론에서 독자에게 질문이나 사색 거리 제시

키워드: 3-5개 (브런치 내 검색용)
```

## Korean Copy Patterns

### 제목 (Title) Patterns

| Pattern | Template | Example |
|---------|----------|---------|
| 숫자 + 결과 | "[숫자]가지 [방법]으로 [결과] 달성하기" | "5가지 방법으로 월 100만원 부수입 만들기" |
| 질문형 | "[대상]이 [상황]할 때 어떻게 하시나요?" | "1인 개발자가 마케팅할 때 어떻게 하시나요?" |
| 비교형 | "[A] vs [B]: [핵심 차이점]" | "노코드 vs 코딩: 비개발자가 선택해야 할 방법" |
| 경험담 | "[기간] 동안 [한 일]에서 배운 [교훈]" | "6개월 동안 SaaS를 만들며 배운 7가지 교훈" |
| 반전형 | "[일반적 믿음]은 틀렸습니다" | "MVP를 만들면 안 됩니다" |

### CTA (행동유도문구) Patterns

Korean CTAs use the ~하기 form:

| English CTA | Korean CTA | Context |
|-------------|------------|---------|
| "Get started" | "시작하기" | General |
| "Try for free" | "무료로 체험하기" | Free trial |
| "Learn more" | "자세히 알아보기" | Information |
| "Sign up" | "가입하기" / "신청하기" | Registration |
| "Download" | "다운로드하기" / "받아보기" | Resources |
| "Contact us" | "문의하기" | B2B |
| "See pricing" | "요금제 확인하기" | Pricing pages |

### 존댓말 (Honorific Level) Guide

| Level | Style | When to Use | Example |
|-------|-------|-------------|---------|
| 해요체 | 편한 존댓말 | B2C, 블로그, SNS | "시작해보세요" |
| 합니다체 | 격식 존댓말 | B2B, 공식 자료, 기업 | "시작하실 수 있습니다" |
| 해라체 | 신문/칼럼체 | 브런치, 에세이, 칼럼 | "시작해 보자" |
| 반말 | 비격식 | 친한 커뮤니티, 밈 | 거의 사용 안 함 |

**Default**: 해요체 for most content. When in doubt, use 해요체.

### Korean-Specific Content Considerations

- **숫자와 데이터**: Koreans respond well to specific numbers ("14,200개 팀" > "많은 팀")
- **사회적 증거**: Korean audiences value group validation more — "국내 1,000개 기업이 사용 중"
- **신뢰 요소**: Certifications, partnerships, press mentions carry extra weight
- **계절/시기**: Korean content calendar differs (Lunar New Year, 추석, 대학입시 시즌 etc.)
- **모바일 우선**: 95%+ of Korean internet usage is mobile — all content must be mobile-first
- **검색 문화**: Koreans search differently — longer, more specific queries on Naver
- **커뮤니티 중심**: Word-of-mouth in Naver Cafe and KakaoTalk groups drives adoption

## Translation vs. Adaptation Checklist

When adapting English content to Korean:

- [ ] Rewrote (not translated) the title for Korean search patterns
- [ ] Changed examples to Korean-relevant ones (Korean companies, Korean market data)
- [ ] Adjusted honorific level consistently throughout
- [ ] Replaced Western idioms/references with Korean equivalents
- [ ] Used Korean CTA patterns (~하기 form)
- [ ] Checked character counts for platform (Korean chars ≈ 2 bytes)
- [ ] Mobile-friendly formatting (short paragraphs, lots of whitespace)
- [ ] Added relevant Korean tags/keywords
- [ ] Cultural references updated (holidays, business customs, trends)
- [ ] Tested readability on mobile (카카오톡 font rendering differs)
